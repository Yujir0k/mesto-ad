# Mesto

Интерактивная страница с карточками мест. Данные загружаются с API: можно
обновлять профиль и аватар, добавлять и удалять свои карточки, ставить лайки и
просматривать статистику карточек по клику на логотип.

## Запуск проекта

```bash
npm install
npm run dev
```

## Сборка и публикация

```bash
npm run build
npm run deploy
```

Опубликованный проект: [Mesto](https://yujir0k.github.io/mesto-ad/)
